.waykeecom-no-margin removes margins
Spacing utility classes are automatically generated classes used to add spacing between elements. Each property can be set on a specific axis; none (all), t (top), b (bottom), l (left), r (right), x (left and right) and y (top and bottom) following this class name format: [NAMESPACE][PROPERTY_ABBR]-[AXIS]--[SIZE]. To cretea a margin bottom with size 5, the class name looks like this: .waykeecom-m-b--5.
